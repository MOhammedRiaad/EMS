## Getting Started

### Setting Up Your Account

1. You'll receive an invitation email from your studio
2. Click the setup link
3. Create your password
4. Complete your profile

### Logging In

1. Navigate to `http://your-studio.com/coach`
2. Enter your email and password
3. Click **Login**

## Managing Today's Sessions

### Viewing Your Day

1. The dashboard shows today's sessions by default
2. Each session card shows:
   - Client name
   - Session time
   - Room assigned
   - Session status

### Session Cards

Each session shows:
- **Client Photo** (if uploaded)
- **Client Name** 
- **Time Slot**
- **Room**
- **Status Badge**

### Filtering Sessions

- Use **Date Picker** to view other days
- Filter by **Status** (Scheduled, Completed, etc.)

Click on a client's name to view:

- **Health Notes**: Any relevant conditions
- **Recent Sessions**: History and notes

### Viewing InBody Data

1. Select the client
2. Go to **Progress** tab
3. Review body composition trends
4. Access previous scan reports

## Completing Sessions

### Mark as Complete

1. At the end of the session, find the session card
2. Click **Complete Session**
3. Add session notes (optional):
   - Intensity level
   - Client feedback
   - Exercises performed
4. Click **Save**

### Marking No-Shows

If a client doesn't arrive:

1. Wait until session time passes
2. Click **Mark No-Show**
3. Confirm the action

> No-shows are handled according to studio policy for session deductions.

### Late Cancellations

If notified of a late cancel:

1. Update session status to **Cancelled**
2. Add a note about the cancellation
3. The admin will handle package adjustments

## Common Questions

### What if a client is late?

Start when they arrive and complete the session as normal. Note the late arrival in session notes if significant.

### How do I handle equipment issues?

1. Report to admin immediately
2. Note the issue in the session
3. Continue with alternative equipment if available

### Can I see my schedule for next week?

Yes! Use the calendar navigation to view any future date. You can see all scheduled sessions in advance.

### How do I contact a client?

Click on the client's name to see their contact information. Always follow your studio's communication guidelines.
